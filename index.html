# Cueva Mamut ![Acceder](https://chatgpt.com/g/g-6a33e10ce6d08191b575c19d7b7389ec-cueva-mamut) ## Política de privacidad Última actualización: 19 de junio de 2026. Este GPT utiliza una acción externa para registrar estadísticas básicas sobre su utilización. Datos recopilados Al iniciarse una conversación mediante el envío del primer mensaje, se puede registrar: Un identificador aleatorio asociado a la conversación. La fecha y hora de inicio. El nombre del usuario únicamente cuando este lo haya proporcionado voluntariamente durante la conversación. Esta acción no recibe automáticamente el nombre de la cuenta, la dirección de correo electrónico ni otros datos de la cuenta de ChatGPT. Finalidad Los datos se utilizan exclusivamente para medir el número de conversaciones iniciadas, analizar el uso general del GPT y mejorar el servicio. Almacenamiento Los datos se transmiten a una aplicación desarrollada con Google Apps Script y se almacenan en una hoja de Google Sheets administrada por el responsable de este GPT. Conservación Los datos se conservarán durante un máximo de [INDICAR PLAZO, por ejemplo: 12 meses], salvo que sea necesario conservarlos durante más tiempo para atender una obligación legal. Compartición de datos Los datos no se venden ni se utilizan con fines publicitarios. Para su tratamiento se utilizan los servicios técnicos de Google necesarios para ejecutar Google Apps Script y almacenar la hoja de cálculo. Derechos y contacto Para solicitar información, corrección o eliminación de los datos asociados, puedes contactar con: Responsable: Ander Frago Correo electrónico: ander_frago@cuatrovientos.org Sitio web: cuatrovientos.org Debido a que el identificador de la conversación es aleatorio, puede ser necesario que el usuario facilite información adicional para poder localizar un registro concreto. Cambios Esta política puede actualizarse cuando cambie el funcionamiento del GPT o de la acción externa.
